# Formation CEMOS Tracteur – ARION 6.190 CMATIC

Site statique compatible avec GitHub Pages.

## Mise en ligne
1. Déposer `index.html` et le dossier `assets` à la racine du dépôt GitHub.
2. Ouvrir **Settings > Pages**.
3. Choisir la branche `main` et le dossier `/ (root)`.
4. Enregistrer.

Le site fonctionne également hors connexion en ouvrant directement `index.html`.


## Version sans surbrillance
Les captures sont affichées sans cadre jaune ni annotation permanente, dans un style proche d’une documentation constructeur.


## Version professionnelle
En-tête épuré : logo CASA unique à gauche, titre centré, boutons Accueil/Sommaire/Continuer à droite.
